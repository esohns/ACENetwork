Weitere Galerien
1/5

Eine Rolle Toilettenpapier liegt bei einem Getränkemarkt in Stuttgart (Baden-Württemberg) auf einem Getränkekasten. Aufgrund von Hamsterkäufen wird weniger Leergut im Mehrweg-System umgetauscht.
